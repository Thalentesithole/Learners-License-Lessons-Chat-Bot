# 🚗 Learners License Lessons Chatbot

An AI-powered chatbot designed to help users study and understand **learner’s license rules, road signs, and driving theory** through interactive conversation.

---

## 📌 Features

- 🤖 AI chatbot for learner’s license Q&A  
- 📚 Retrieval-based responses using a vector database  
- 🧠 Context-aware answers using LLMs  
- 🔍 Semantic search with embeddings  
- 🌐 Simple web interface (Flask)  
- ☁️ Deployable with Docker & CI/CD  

---

## 🏗️ Tech Stack

- **Backend:** Flask (Python)  
- **AI/LLM:** OpenAI / LangChain  
- **Vector Database:** Pinecone  
- **Embeddings:** HuggingFace / OpenAI  
- **DevOps:** Docker, GitHub Actions  

---

## 🧠 How It Works
- Learning material is split into chunks
- Text is converted into embeddings
- Stored in Pinecone vector database
- User query is embedded and matched
- Relevant context is retrieved
-LLM generates a response
